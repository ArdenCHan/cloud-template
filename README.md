# SpringCloud分布式微服务项目

## 介绍

旨在提供一个最基础的CRUD模板，并没与太复杂的业务流程，想让初学微服务的同学有一个案例参考。大家可以基于此项目继续扩展，或者看此项目和源码和文档自己尝试开发前后端分离的项目。
